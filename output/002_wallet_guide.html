1부

디지털 월렛이란 무엇인가

월렛은 코인을 담는 '주머니'가 아니라, 블록체인에 기록된 자산을 움직일 권한(키)을 관리하는 도구입니다. 먼저 이 핵심 발상부터 잡고 시작합니다.

1. 한 문장 정의

디지털 월렛은 블록체인에 기록된 내 디지털 자산을 확인·전송·사용할 수 있도록 '암호학적 열쇠'를 관리하는 도구입니다.

핵심 포인트

코인이나 NFT가 월렛 앱 안에 파일처럼 저장되는 것이 아닙니다. 자산과 거래 내역은 블록체인에 기록되고, 월렛은 그 자산을 움직일 권한인 개인키를 관리합니다.

2. 쉬운 비유: 월렛은 지갑보다 '열쇠 꾸러미'에 가깝다

아파트를 생각해 보겠습니다.

비유실제 개념
공동 장부블록체인 — 모든 집과 소유·이동 기록이 적힌 장부
우리 집 공개 주소지갑 주소 — 남에게 알려줘도 되는 식별값
문을 여는 비밀 열쇠개인키 — 거래에 서명하는 권한
열쇠 꾸러미·도어록 앱디지털 월렛 — 주소와 열쇠를 관리
장부상 보유 자산코인·토큰·NFT

누군가 내 지갑 주소를 아는 것은 집 주소를 아는 것과 비슷합니다. 주소만으로는 자산을 이동시킬 수 없습니다. 하지만 개인키를 가진 사람은 거래에 디지털 서명을 할 수 있으므로, 사실상 그 지갑을 통제합니다.

3. 디지털 월렛은 왜 필요한가

대표적으로 다음 일을 합니다.

자산 확인

내 주소가 보유한 코인·토큰·NFT를 조회합니다.

자산 수신

상대방에게 지갑 주소를 알려주고 자산을 받습니다.

자산 전송

받는 주소와 수량을 입력한 뒤 개인키로 거래에 서명합니다.

블록체인 서비스 이용

거래소·게임·NFT 서비스 등에 연결해 로그인하거나 거래를 승인합니다.

월렛은 단순한 자산 목록 화면이 아니라, 블록체인에 특정 행동을 요청하고 그 요청에 서명하는 인터페이스입니다.

보완 NIST의 월렛 정의 (출처 보강)

NIST의 블록체인 간행물 NISTIR 8301 "Blockchain Networks: Token Design and Management Overview"(2021)는 ISO 22739:2020 표준 정의를 인용해, 월렛을 "개인키와 공개키를 생성·관리·보관·사용하는 애플리케이션"(an application used to generate, manage, store or use private and public keys)으로 설명합니다.[1]

출처: NIST (미국 국립표준기술연구소), NISTIR 8301 (2021) · ISO 22739:2020 인용 · 확인 2026-06-26

4. 일반 금융 앱과 무엇이 다른가

구분은행·금융 앱일반적인 블록체인 월렛
자산 기록 장소은행·금융회사 서버블록체인 네트워크
계좌 통제 방식금융회사가 계좌를 운영개인키 보유자가 거래를 승인
사용자 식별이름·휴대전화·신분증 등지갑 주소와 암호학적 서명
비밀번호 분실본인 확인 후 재설정 가능개인키·복구 수단을 잃으면 복구가 어려울 수 있음
잘못된 송금기관을 통한 조치 가능성중앙기관이 없어 되돌리기 어려운 경우가 많음
서비스 연결해당 금융회사 기능 중심여러 블록체인 앱에 연결 가능

다만 모든 디지털 월렛이 사용자가 직접 개인키를 관리하는 것은 아닙니다. 회사가 대신 관리하는 월렛도 있습니다. 이 차이를 커스터디(Custodial)와 논커스터디(Non-custodial)라고 합니다.

5. 지금 알아야 할 핵심 용어

블록체인
여러 참여자가 함께 유지하는 거래 장부. 한 회사의 중앙 서버만 믿는 대신, 네트워크가 정해진 규칙에 따라 거래의 유효성을 확인하고 기록합니다.
네트워크 · 체인
각자 다른 규칙과 장부를 가진 블록체인 시스템. 한 월렛이 모든 블록체인을 자동 지원하는 것은 아니므로, 월렛을 분석할 때는 어떤 체인을 지원하는지 확인해야 합니다.
지갑 주소
자산을 받을 때 상대방에게 알려주는 공개 식별값. 은행 계좌번호와 비슷하지만 주소 형식·사용 규칙은 블록체인마다 다를 수 있습니다.
공개키와 개인키
공개키는 주소를 만들고 서명을 검증하는 공개 정보, 개인키는 거래에 서명하고 자산 통제 권한을 증명하는 비밀 정보입니다. 개인키는 단순한 로그인 수단이 아니라 '자산을 움직일 권한' 자체에 가깝습니다.
시드 문구
여러 단어로 이루어진 지갑 복구 정보. 개인키들을 다시 만들어내는 출발점이므로, 노출되면 지갑 통제권을 잃을 수 있습니다.
커스터디와 논커스터디
커스터디 월렛은 회사·서비스가 사용자의 개인키를 대신 관리(은행 금고에 맡기는 방식), 논커스터디 월렛은 사용자가 개인키 또는 복구 권한을 직접 관리(집 금고 열쇠를 직접 보관)합니다. 논커스터디는 통제권이 크지만 복구 정보 관리 책임도 커집니다.
⚠ 보안 경고

지갑 주소는 공개해도 되지만, 개인키와 시드 문구는 누구에게도 알려주면 안 됩니다. 노출되는 순간 지갑 통제권을 잃을 수 있습니다.

6. 월렛 서비스를 처음 볼 때 확인할 세 가지

제품 화면보다 먼저 다음 질문을 확인해야 합니다.

  1. 개인키를 누가 관리하는가? — 사용자, 서비스 회사, 또는 여러 주체가 나누어 관리하는지 확인합니다.
  2. 어떤 블록체인을 지원하는가? — 지원 체인에 따라 이용할 수 있는 코인·NFT·서비스가 달라집니다.
  3. 휴대전화를 잃거나 비밀번호를 잊었을 때 어떻게 복구하는가? — 시드 문구·클라우드 백업·하드웨어 장치·소셜 복구 등 방식이 다릅니다.

이 세 가지가 이후 MetaMask · Phantom · 갤럭시 월렛 · D'CENT · 마이비를 비교하는 기본 축이 됩니다.

오늘의 복습 · 확인 문제

오늘의 복습

  • 월렛 안에 코인이 직접 들어 있는 것은 아니다.
  • 자산 기록은 블록체인에 있고, 월렛은 자산을 움직이기 위한 키와 서명을 관리한다.
  • 지갑 주소는 공개해도 되지만 개인키와 시드 문구는 공개하면 안 된다.
  • 월렛 분석의 출발점은 지원 체인, 키 관리 주체, 복구 방식이다.
다음 과제

아래 상황을 한 문장으로 설명해 보세요. "휴대전화를 잃어버렸지만 시드 문구는 안전하게 보관하고 있다." — 핵심은 자산이 사라진 것인지, 아니면 다른 기기에서 지갑을 복구할 수 있는지 구분하는 것입니다.

예시 답안 보기

Q2. 지갑 주소와 개인키 중 다른 사람에게 알려줘도 되는 것은 무엇인가요?

1. MetaMask

🔌Web3 서비스에 연결하는 범용 소프트웨어 월렛 (개념도 · alt: Web3 범용 소프트웨어 월렛)

쉬운 비유: 여러 Web3 서비스에 들어갈 때 사용하는 로그인 수단이자 개인 금고 열쇠 앱입니다.

브라우저 확장 프로그램과 모바일 앱으로 제공되며, 사용자가 자신의 자산과 접근 권한을 통제하는 셀프 커스터디 월렛입니다.

주요 목적

  • 코인·토큰 보관·전송
  • 탈중앙화 거래소·NFT 마켓 등 Web3 서비스 연결
  • 거래와 스마트 계약 요청에 서명
보완 지원 체인 최신 정보 (2026-06 기준)

MetaMask 공식 자료에 따르면 네이티브로 Ethereum, Linea, Base, Polygon, BNB Chain, Arbitrum, OP Mainnet, Avalanche, zkSync Era, Sei, Bitcoin, Solana, TRON 등을 지원합니다. Solana 네이티브 지원은 2025년 7월, Bitcoin 네이티브 지원은 2025년 12월에 추가되었습니다(이전에는 Snaps 확장으로만 가능).

출처: MetaMask 공식 FAQ / Help Center[2] · 지원 체인은 수시로 변경되므로 분석 시점에 공식 페이지 재확인 필요

2. Phantom

👻솔라나에서 출발해 멀티체인으로 확장한 사용성 중심 월렛

쉬운 비유: MetaMask와 비슷한 열쇠 앱이지만, 처음에는 솔라나 전용 출입증에 가까웠고 지금은 여러 건물에 들어갈 수 있도록 확장된 형태입니다.

Phantom은 사용자가 자산 통제권을 가지는 셀프 커스터디 방식의 멀티체인 월렛입니다.

주요 목적

  • 코인·토큰 관리
  • NFT 확인 및 관리
  • Web3 앱 연결
  • 월렛 안에서 자산 전송·교환

MetaMask와의 기본 차이

  • MetaMask는 이더리움·EVM 생태계에서 강하게 성장
  • Phantom은 솔라나 생태계에서 강하게 성장
  • 현재는 둘 다 멀티체인 방향으로 확장 중
보완 지원 체인 최신 정보 (2026-06 기준)

Phantom 공식 헬프센터(2026-06-09 갱신) 기준 지원 네트워크는 Solana, Ethereum, Base, Polygon, Sui, Monad, Bitcoin(Native SegWit/Taproot), HyperEVM이며, BSC·Arbitrum·Optimism·Avalanche·Linea 등은 미지원입니다.

3. 갤럭시 월렛 관련 추가 확인 필요

📱스마트폰 제조사가 기기 보안과 결합해 제공하는 월렛 생태계
가정

여기서 '갤럭시'는 삼성의 Samsung Wallet, Samsung Blockchain Wallet 및 갤럭시 기기 내 보안 기능을 통칭합니다. 하나의 앱만을 뜻하지 않습니다.

Samsung Wallet

결제 카드·멤버십·신분증·탑승권·자동차/주택 디지털 키 등을 한곳에서 관리하는 생활형 디지털 월렛입니다. 일반 암호화폐 월렛보다 범위가 넓으며, 2026년에도 삼성은 결제·여행·디지털 키 기능을 확장하고 있습니다.

Samsung Blockchain Wallet

블록체인 자산을 확인·거래할 수 있도록 갤럭시 기기와 연결된 암호화폐 월렛 기능입니다.

보완 비교 환경 범위 명시 (부분 보완)

(1) Samsung Wallet은 갤럭시 기기 기반으로 국가별 단계적 제공되며 결제·여행·디지털 키 영역이 확장 중입니다. (2) Samsung Blockchain Wallet은 공식 지원 문서 기준 Galaxy S10 시리즈 이후의 S/Note/Fold/Z Flip 시리즈와 일부 A 시리즈(2020년 이후)에서 동작하며, 미국(Verizon 제외)·유럽 12개국·싱가포르·필리핀·호주 등 제한된 국가에서만 제공됩니다.

따라서 비교 기준은 '특정 갤럭시 단말 + 지원 국가 + 해당 OS 버전' 환경으로 한정해야 합니다.

출처: Samsung 공식 지원 페이지(미국/싱가포르)[4]

⚠ 추가 확인 필요

한국 내 Samsung Blockchain Wallet의 정식 지원 여부와 잔존 기능 범위는 Samsung 공식 한국 페이지에서 별도 확인이 필요합니다.

4. D'CENT

🔐인터넷과 분리된 하드웨어 콜드 월렛

쉬운 비유: MetaMask·Phantom이 휴대전화 안의 열쇠 앱이라면, D'CENT는 중요한 열쇠를 별도의 전자 금고나 보안 카드에 넣어두는 방식입니다.

D'CENT는 지문 인증 하드웨어 월렛, NFC 카드형 월렛, 모바일 앱 월렛을 제공합니다. 모바일 앱은 자산 현황을 보여주고 하드웨어 기기·카드와 연결하는 인터페이스 역할을 합니다.

주요 목적

  • 개인키를 인터넷과 분리된 장치에서 보호
  • 큰 금액·장기 보유 자산 관리
  • 하드웨어 장치에서 거래 확인·서명
  • 모바일 앱에서 자산 조회와 서비스 이용

MetaMask·Phantom과의 기본 차이

  • MetaMask·Phantom: 앱 중심의 편리한 사용
  • D'CENT: 별도 하드웨어를 통한 키 보호가 핵심
  • 편의성은 앱 월렛이, 물리적 분리 보안은 하드웨어 월렛이 강점

5. 마이비(MyB) 추가 확인 필요

🎁블록체인을 의식하지 않아도 되는 리워드 내장형 임베디드 월렛

쉬운 비유: 블록체인 월렛 자체라기보다, 퀴즈·걷기·쇼핑 등으로 포인트를 모으는 리워드 앱에 블록체인 지갑 기능을 연결한 서비스에 가깝습니다.

그래서 MetaMask·Phantom 같은 범용 Web3 월렛과 동일하게 보면 안 됩니다.

  • MetaMask·Phantom: 월렛이 중심
  • 마이비: 리워드 서비스가 중심이고 블록체인이 내부 기능으로 결합
보완 확인된 사실 (부분 보완)

마이비(MyB)는 블록체인 스타트업 원셀프월드(1Self.World)가 운영하는 리워드(앱테크) 서비스로, 2025년 1월 XPLA 메인넷에 온보딩되었습니다. 모든 트랜잭션, SBT(소울바운드 토큰) 발행, 디지털 지갑 관리는 XPLA 네트워크에서 처리됩니다. 출시 6개월 만에 가입자 55만 명을 기록했고, 퀴즈·만보기·게임·쇼핑·설문으로 포인트를 적립해 현금화하는 구조입니다.

출처: ZDNet Korea, 디지털투데이, 마이비 공식 사이트[5]

⚠ 추가 확인 필요 (공개 자료로 확정 불가)

마이비의 ① 개인키 커스터디 주체(사용자/원셀프월드/XPLA 중 누가 보관하는지), ② 구체적 지갑 복구 방식(시드 문구·소셜 복구 등), ③ 외부 지갑으로의 자산 이동(출금) 허용 범위는 공개 자료만으로 확정하기 어렵습니다.

전체 구도

다섯 키워드는 같은 종류의 경쟁 서비스가 아닙니다. 유형별로 분류하면 다음과 같습니다.

범용 소프트웨어 월렛
 ├─ MetaMask
 └─ Phantom

하드웨어 중심 월렛
 └─ D'CENT

단말·생활 플랫폼 월렛
 └─ 갤럭시 / Samsung Wallet 생태계

리워드 서비스 내장형 Web3
 └─ 마이비(MyB)
분석 원칙

유사 서비스 분석에서도 단순히 기능 수를 비교하면 안 됩니다. 먼저 '개인키를 누가 관리하는가 · 어떤 체인을 지원하는가 · 어떻게 복구하는가'라는 질문으로 서비스의 정체성(유형)을 구분해야 합니다.

용어 사전

본문에 등장한 약어·전문 용어 풀이입니다.

EVM (Ethereum Virtual Machine)
스마트 계약을 실행하는 이더리움의 표준 실행 환경. 이를 호환하는 체인(Polygon, BNB Chain 등)을 'EVM 계열'이라 부릅니다. MetaMask가 강점을 가진 생태계입니다.
Web3
블록체인·암호화폐 기반의 탈중앙화 인터넷. 사용자가 중앙 서버 대신 자신의 지갑(키)으로 신원을 증명하고 자산을 직접 통제하는 서비스 환경을 뜻합니다.
XPLA (엑스플라)
컴투스홀딩스 진영이 주도하는 글로벌 블록체인 메인넷. 게임·콘텐츠·리워드 등 다양한 dApp이 온보딩되어 있으며, 마이비(MyB)의 트랜잭션과 지갑 관리가 이 네트워크에서 처리됩니다.
콜드 월렛 (Cold Wallet)
개인키를 인터넷과 분리된 오프라인 환경(전용 하드웨어·보안 카드 등)에 보관하는 월렛. 온라인 '핫 월렛'보다 해킹 노출이 적습니다. D'CENT가 대표 예입니다.
NFC (Near Field Communication)
기기를 가까이 대면 데이터를 주고받는 근거리 무선 통신 기술. D'CENT의 카드형 월렛이나 비접촉 결제에 사용됩니다.
임베디드 월렛 (Embedded Wallet)
사용자가 시드 문구·개인키 같은 블록체인 개념을 직접 다루지 않아도 앱 내부에 자동으로 내장·연동되는 지갑. 마이비처럼 리워드 앱 안에 블록체인 기능이 숨어 있는 형태입니다.
SBT (소울바운드 토큰)
양도·복제가 불가능하도록 특정 지갑에 귀속된 토큰. 마이비는 퀴즈 등으로 발급되는 '취향 뱃지'를 SBT로 발행합니다.
셀프 커스터디 (Self-Custody)
사용자가 자신의 개인키 또는 복구 권한을 직접 보관·관리하는 방식. 논커스터디와 같은 의미로 쓰이며 MetaMask·Phantom이 해당합니다.
Snaps
MetaMask의 확장 기능 시스템. 코어가 네이티브로 지원하지 않는 체인(NEAR, Cosmos, Tezos, Cardano 등)을 서드파티 플러그인으로 추가 연결할 수 있게 합니다.
